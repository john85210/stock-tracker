📤 如何把變更同步到 GitHub

你剛剛下載了 data.json。把它上傳回 GitHub 後，任何人用任何瀏覽器（包含無痕視窗、手機、別台電腦）打開網頁都會看到最新資料。

  1. 到 你的 stock-tracker repo
  2. 如果是第一次：點右上角 Add file → Upload files，把剛下載的 data.json 拖進去
  3. 如果已經存在：點進 data.json → 右上鉛筆圖示 → 全選刪掉 → 貼上新檔內容
  4. 滾到最下面按 Commit changes
  5. 等 1~2 分鐘讓 GitHub Pages 刷新
  6. 重新整理網頁即可看到同步後的狀態

💡 最快的方式：直接把 data.json 拖到 GitHub repo 的網頁上，它會自動進入 commit 流程。
